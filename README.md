# MiniCursoFinalProject

This  is the final project used as the support material for one of my workshops. The app goes over the following topics:

* Swift
* XCode
* iOS
* App Delegate
* Storyboard 
* Segue
* Constraints
* Delegates
* Keyboard Notifications
* UITextField
* UITableView
* UINavigationController
* UIAlertView
* UITabBarController
